# ArchiMate (archimate)
ArchiMate is an open and independent enterprise architecture modeling language  to support the description, analysis and visualization of architecture within  and across business domains in an unambiguous way.


**URL:** [Visit APIs.json URL](https://www.opengroup.org/archimate-forum)

## APIs

### ArchiMate Model Exchange API
API for exchanging ArchiMate models between tools and repositories using  the Open Group ArchiMate Model Exchange File Format.


**Human URL:** [https://www.opengroup.org/archimate-forum/archimate-overview](https://www.opengroup.org/archimate-forum/archimate-overview)


#### Tags:

 - Model Exchange, XML, Enterprise Architecture

### ArchiMate Repository API
RESTful API for accessing and managing ArchiMate models, elements,  relationships, and views stored in a central repository.
